:active - Applies to an element when it is being activated by the user, for example when clicking a button.
:checked - Applies to checkboxes and radio buttons when they are checked.
:default - Applies to form controls when they are set to their default values.
:disabled - Applies to form controls that are disabled.
:empty - Applies to elements that have no children or content.
:enabled - Applies to form controls that are enabled.
:first-child - Applies to the first child element of a parent element.
:first-of-type - Applies to the first element of a particular type within a parent element.
:focus - Applies to an element when it has focus, for example when clicking into an input field.
:hover - Applies to an element when the user hovers over it with the mouse.
:indeterminate - Applies to checkboxes and radio buttons when they are in an indeterminate state.
:in-range - Applies to input elements with a "range" type that have a value within a specified range.
:invalid - Applies to input elements that have an invalid value.
:last-child - Applies to the last child element of a parent element.
:last-of-type - Applies to the last element of a particular type within a parent element.
:link - Applies to unvisited links.
:not() - Applies a style to an element that does not match the specified selector(s).
:nth-child() - Applies to every nth child element of a parent element.
:nth-last-child() - Applies to the nth child element of a parent element, counting from the end.
:nth-last-of-type() - Applies to the nth element of a particular type within a parent element, counting from the end.
:nth-of-type() - Applies to every nth element of a particular type within a parent element.
:only-child - Applies to elements that are the only child element of a parent element.
:only-of-type - Applies to elements that are the only element of a particular type within a parent element.
:optional - Applies to form controls that are not required.
:out-of-range - Applies to input elements with a "range" type that have a value outside a specified range.
:read-only - Applies to form controls that are read-only.
:read-write - Applies to form controls that are read-write.
:required - Applies to form controls that are required.
:root - Applies to the root element of the document.
:target - Applies to the element with the ID specified in the URL fragment identifier.
:valid - Applies to input elements that have a valid value.
:visited - Applies to visited links.
